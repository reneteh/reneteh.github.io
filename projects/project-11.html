Coffee Subscription Service

Creative Web Design, HTML & CSS — 2025

This project began with the intention of designing a coffee subscription website that appeals to a younger audience through a more playful and engaging visual identity. Early explorations focused on moodboarding and logo development, where I experimented with illustrative styles, bold colours, and expressive typography to create a fun and approachable brand language. Through iterations, I moved away from rigid and overly structured visuals towards a more fluid and character-driven direction, establishing a balance between modern aesthetics and a lighthearted tone.

The design process was guided by structure and clarity, beginning with wireframes that mapped out the key pages—home, about, subscription, and download. Initial layouts felt overly complex and repetitive, which led to a refinement towards cleaner grid-based compositions that combined photography with illustrations. Each page was designed with a consistent system, such as the use of full-bleed header images and column layouts, while adapting where necessary, for example, introducing a three-column structure in the subscription page to clearly differentiate between tiers. This iterative process emphasised the importance of visual hierarchy, consistency, and user flow, ensuring that the interface remained intuitive while still visually engaging.

The final outcome is a responsive website built using HTML, CSS, and JavaScript, integrating both visual design and interactive elements. Enhancements such as hover effects, marquee animations, and parallax scrolling were introduced to create a more dynamic user experience, while maintaining simplicity to match the overall aesthetic. Despite challenges with alignment, responsiveness, and layout behaviour across different screen sizes, these issues were resolved through experimentation and refinement. Ultimately, the project demonstrates how a strong visual identity, combined with thoughtful interaction design, can transform a functional service into an engaging and cohesive digital experience.
